Backend project of an online store created with GraphQL.

## Technologies

- Nestjs v10
- Docker
- PostgreSQL
- GraphQL
- Apollo Server

## Development

1. Clone the project
2. Copy `.env.template` file and rename to `.env`
3. Execute command

```
yarn install
```

4. Run docker image

```
docker-compose up -d
```

5. Start project

```
yarn start:dev
```
